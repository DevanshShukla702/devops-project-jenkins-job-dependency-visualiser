FlowTrace

Jenkins Job Dependency Visualizer

📘 About The Project

FlowTrace transforms Jenkins pipelines from standard text logs into fully interactive, auto-updating dependency graphs. It is designed to solve the complexity of large scale CI/CD setups where tracing upstreams, downstreams, and build failures takes minutes.

Built as a professional portfolio tool and scalable utility for DevOps engineers, FlowTrace seamlessly sits on top of any Jenkins instance via REST API.

Tech Stack Built With:

Python 3.11 Flask vis-network Vanilla JS GSAP Jenkins REST API
